# Parametrized QED (pQED) Many-Body Dispersion (MBD)
Collection of scripts to run the parametrized quantum electrodynamics (pQED) approach based on time-dependent density functional theory (TDDFT) for non-covalently interacting systems, including van der Waals and hydrogen bonding interactions used in the paper: 

["Parameterized Molecular Cavity Quantum Electrodynamics Approach to Simulate van der Waals Interactions"](https://chemrxiv.org/engage/chemrxiv/article-details/693b2d6eef27c95d3b4520bf)  

Check $\Rightarrow$ https://chemrxiv.org/engage/chemrxiv/article-details/693b2d6eef27c95d3b4520bf

---
Each folder contains the initial geometry (Gaussian16 input file), output data (Data folder) and plotting script for the systems:
- Argon Dimer
- Water Dimer
- Benzene Dimer
- Benzene Argon system

The codes used to generate the results in the paper are available in the *PolaritonCodes* folder.

For more detail information on these codes, check *Braden M. Weight* repository:
- https://github.com/bradenmweight/Ab_Initio_Polariton_Properties
